# Nike Sales Performance Analytics Project

## Project Overview
This project is a data-driven analysis of Nike sales performance using Python and Jupyter Notebook. The goal is to analyze sales, revenue, and profit trends and derive meaningful business insights.

## Objectives
- Analyze sales performance
- Evaluate revenue trends
- Study profit distribution
- Identify key business insights

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Source: Kaggle

## Project Files
- Final Project Nike Sale Performance Analytics.ipynb
- Nike Sales Performance Analytics.csv
